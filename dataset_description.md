# 데이터셋 구성표 (Dataset Description)

본 문서는 모델 학습에 사용되는 자체 구축 영상 데이터셋의 전체 통계 및 분포를 기록하는 문서입니다.

## 1. 데이터셋 총괄 요약
- 데이터셋 구축 담당자: 문지민
- 총 영상 개수: 80개 
- 영상 포맷: MOV 및 MP4

## 2. 클래스별 데이터 분포
데이터셋의 클래스 불균형 문제를 방지하기 위해 균등한 비율로 수합되었습니다.

| 클래스명 (Class) | 영상 개수 (Count) | 데이터 비율 (%) | 데이터 정량적 특징 요약 |
| :--- | :---: | :---: | :--- |
| **normal** | OO 개 | 37.5% | 짧은 체류 시간 (대부분 8초 미만), 일방향 이동 동선 |
| **suspicious** | OO 개 | 31.25% | 긴 체류 시간 (8초 이상), 복도 내 왕복 이동 |
| **dangerous** | OO 개 | 31.25% | 문 앞 집중 체류 (80% 이상), 도어락/문고리 마찰 발생 |
| **합계** | **OO 개** | **100%** | **총 특징 데이터 추출 준비 완료** |

## 3. 디렉토리 구조 (Directory Structure)
현재 프로젝트 루트 기준 데이터 저장 구조는 다음과 같습니다. (팀의 실제 폴더 구조에 맞게 수정 가능)
```text
osp_project/
├── normal/            # 일반 상황 영상 파일
├── suspicious/        # 의심 상황 영상 파일
├── dangerous/         # 위험 상황 영상 파일
├── labels.csv         # 영상 경로와 라벨 매핑 파일 (지민 완성본)
└── docs/
    ├── label_rules.md
    └── dataset_description.md
